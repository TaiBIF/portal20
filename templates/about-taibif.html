臺灣生物多樣性資訊機構
(Taiwan Biodiversity Information Facility, TaiBIF)

{% trans "關於 TaiBIF" %}

{% url 'about_gbif' as about_gbif %} {% blocktrans %} 臺灣生物多樣性資訊機構 (Taiwan Biodiversity Information Facility, TaiBIF) 是全球生物多樣性資訊機構 (GBIF) 在臺灣的國家節點，也是臺灣生物多樣性資訊與國際平台接軌之窗口。 GBIF.org 國際平台上，所有授權條件明確之公開生物多樣性資料，均提供全球科學研究、政策制定以及公共建設使用。 TaiBIF 於 2004 年由國科會委託中央研究院生物多樣性研究中心成立及運作，主要任務為推動、建置臺灣生物多樣性資訊相關基礎建設， 包含生物多樣性資訊的開放分享、研究資料管理流程的建置與交流，以及資訊服務的設計與開發。 2008 年 3 月，TaiBIF 於中央研究院成立全球生物多樣性資訊機構中華民國委員會 (GBIF-ROC)，促進跨部會在生物多樣性資訊議題上的對話，以推動資訊的流通與整合。 {% endblocktrans %}

{% trans "目標及任務" %}

{% trans "我們致力於推動生物多樣性資訊的整合、管理、流通及應用，並包含兩個層面：" %}

  1. {% blocktrans %} 生物多樣性資訊服務及基礎建設的開發，透過制度、流程的建立及軟體工程方法，確保資料的發布和取用流程中，均有合適的軟體協助解決方案。 {% endblocktrans %}
  2. {% blocktrans %} 資料共享的文化，透過與各主管單位及研究團隊討論資料管理的觀念和方法，提升相關人員對於資料開放價值的認同，從而願意將資料的整理、備份，以及資料標準化、加強資料可讀與再利用性等工作，納入日常研究活動當中。 {% endblocktrans %}

{% trans "根據上述的目標，本專題中心主要有下列任務：" %}

  • {% trans "建置生物多樣性資訊基礎建設" %}
  • {% trans "推廣開放資料的概念" %}
  • {% trans "促進生物多樣性資料的開放與整合" %}
  • {% trans "支持生物多樣性科學研究與政策制定" %}
  • {% trans "連結臺灣與國際的生物多樣性資訊" %}
