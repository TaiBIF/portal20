說明 Usage

最後修改：2023/05/17

使用規範：

本 API 採「政府資料開放授權條款－第1版」

回傳格式說明：

  • 回傳格式為 JSON。
  • 查詢網址為 https://portal.taibif.tw/api/{version}/{type}/{mode}?{parameter}，其中：
    • {version}：API 版本，此版本為 v2 。
    • {type}：查詢資料種類，目前提供 occurrence、dataset 與 publisher 。
    • {mode}：限出現紀錄 (occurrence) 資料類型查詢，目前提供 basic_occ（轉譯欄位 + 基本欄位資料回傳）、 detail_occ（轉譯欄位 + 基本欄位 + 常用欄位資料回傳）與 raw_occ（原始資料回傳）。
      各自的差別請看出現紀錄查詢類型說明。
    • {parameter}：查詢參數
    • 網址範例：
    資料種類{type}範例
    occurrence (basic_occ) https://portal.taibif.tw/api/v2/occurrence/basic_occ?{parameter}=
    occurrence (detail_occ) https://portal.taibif.tw/api/v2/occurrence/detail_occ?{parameter}=
    occurrence (raw_occ) https://portal.taibif.tw/api/v2/occurrence/raw_occ?{parameter}=
    dataset https://portal.taibif.tw/api/v2/dataset?{parameter}=
    publisher https://portal.taibif.tw/api/v2/publisher?{parameter}=

  • 分頁參數說明：

    參數名稱 {parameter} 說明範例
    rows每頁回傳筆數。預設為 100 筆，1000 筆為上限 https://portal.taibif.tw/api/v2/occurrence/basic_occ?rows=100
    https://portal.taibif.tw/api/v2/occurrence/detail_occ?rows=100
    offset 起始回傳資料位置。預設為 0 。 https://portal.taibif.tw/api/v2/occurrence/basic_occ?offset=0
    • 如有建議或額外需求，歡迎聯絡我們。

出現紀錄 Occurrence API

查詢方式：

用途參數名稱 {parameter}查詢網址
取得出現紀錄資料請看回傳欄位說明/api/v2/occurrence/

出現紀錄查詢類型

  • 出現紀錄資料類型查詢 {mode} 說明：
    • 定義名稱描述欄位名稱
    轉譯資料TaiBIF 針對特定欄位資料做清理 (例如統一格式等)共 33 個欄位：taibifOccurrenceID、scientificName、isPreferredName、taxonRank、taicolTaxonID、gbifAcceptedID、scientificNameID、taxonBackbone、eventDate、decimalLongitude、decimalLatitude、geodeticDatum、verbatimSRS、coordinateUncertaintyInMeters、basisOfRecord、selfProduced、license、occurrenceStatus、kingdom、phylum、class、order、family、genus、taxonGroup、country、countryCode、county、year、month、day、habitatReserve、wildlifeReserve
    基本資料出現紀錄相關的重要欄位共 17 個欄位：occurrenceID、dataGeneralizations、coordinatePrecision、locality、organismQuantity、organismQuantityType、datasetName、datasetShortName、typeStatus、recordedBy、recordNumber、preservation、collectionID、taibifCreatedDate、taibifModifiedDate、associatedMedia、mediaLicense
    常用資料出現紀錄相關的次要欄位共 4 個欄位：modifiedDate、gbifDatasetID、establishmentMeans、issue
    原始資料資料上傳者所提供的資料
  • 各查詢類型的回傳欄位說明：
    • 類型 {mode}回傳內容回傳欄位
    basic_occ轉譯資料 + 基本資料共回傳 50 個欄位
    detail_occ轉譯資料 + 基本資料 + 常用資料共回傳 54 個欄位
    raw_occ原始資料資料上傳者所提供的所有資料
