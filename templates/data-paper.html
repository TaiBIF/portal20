{% trans "關於資料論文" %}

{% trans "什麼是資料論文？" %}

{% trans "資料論文（Data paper）是一個透過同儕審查（Peer reviewed）的論文形式發表於相關的期刊上。" %} {% trans "與傳統的研究論文不同，資料論文不提假設和結論，" %} {% trans "內容主要為描述作者所開放的資料集相關資訊，" %} {% trans "包含如何產生及收集資料、資料類型、涵蓋的物種、資料開放於何處及此資料集背景資訊（Metadata）等等，" %} {% trans "透過作者於此資料集所開放的資料授權方式，可作為日後學術論文應用其資料時所引用的依據。" %}


{% trans "發表資料論文的好處與目的包括：" %}

  • {% trans "提供可引用的期刊出版物（如DOI），為資料發布者帶來學術上的認可" %}
  • {% trans "人類可讀的結構化資料描述" %}
  • {% trans "增加您發布的資料資源的可見性、可用性和可信度" %}
  • {% trans "更有效地追蹤您發布的資料的使用情況和引用情況" %}

{% trans "資料論文發表指引" %}

{% trans "大致上，您的資料集將決定您適合投稿的主題性資料期刊。" %} {% blocktrans %} 您也可以選擇主題範圍較廣泛且與生物多樣性領域相關的資料期刊「Biodiversity Data Journal」， 或則不限領域限制的「Data in Brief」或「Scientific Data」等期刊（更多期刊資訊請參閱資料期刊）。 此頁將特別介紹arpha Writing Tool的資料論文撰寫與發表工具。 {% endblocktrans %}


{% blocktrans %} arpha Writing Tool是由Pensoft出版社開發的文章發表工具，幫助簡化了發表資料論文的流程。 {% endblocktrans %} {% trans "當您透過TaiBIF的資料發布工具IPT上傳您的資料並發布後，可下載其自動產生的RTF/EML檔案（含GBIF資料集metadata資訊）並上傳至工具裡，此工具會自動將相關資訊填補到適合的文章章節裡。" %} {% blocktrans %} 整個合作撰寫、建議、修改、提交、同儕審查以及發布的動作都能在此線上工具中完成。arpha Writing Tool目前支持直接將資料論文提交至Biodiversity Data Journal。 {% endblocktrans %} {% trans "下列的PDF講義帶領大家一步一步的操作了解資料集如何通過此工具發布資料文章。" %}
