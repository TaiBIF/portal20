什麼是「資料產品」？

一份好的資料集，必須可以再利用（具互操作性），若能提供越多欄位資料，則資料的品質及價值也能相對地提高，在任何情況下，資料集至少要有下列所述 Darwin Core 欄位

相關參考連結：

  • DwC 資料欄位的各集合內容：Darwin Core Registered Extensions（按 core 和 extension 分類）
  • DwC 資料欄位標準英文定義快速查詢：Darwin Core Quick Reference Guide（按詞條的 class 分類）
